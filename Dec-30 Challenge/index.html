This week you are tasked with creating a
function that takes a number as an argument
and returns the appropriate error message.
You should do this without using the switch or if statements.

The input error will be 1 to 5:

1 >> "Check the fan"
2 >> "Emergency stop"
3 >> "Pump Error"
4 >> "Air leakage"
5 >> "Temperature Sensor Error"

For any other value, return 101 (you can use an if statement here).
