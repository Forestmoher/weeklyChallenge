Challenge 1
You probably know the "like" system from
Facebook and other pages. People can
"like" blog posts, pictures, or
other items. We want to create the text
that should be displayed next to such an item.

Implement a function, which takes an
input of an array, containing the
names of people who like an item. It must
return the display text as shown in the examples:

likes(['John','Mary','Jane']) => 'John, Mary, and Jane like this'
likes(['John','Mary']) => 'John and Mary like this'
likes([]) => 'No one likes this'

Challenge 2
Given n, take the sum of the digits of n.
If that value has more than one digit,
continue reducing in this way
until a single-digit number is produced.
The input will be a non-negative integer.

Example:
sumOfDigits(16) => 7 because 1 + 6 = 7
sumOfDigits(567) => 9 becasue 5 + 6 + 7 = 18 and 1 + 8 = 9
Good luck and happy coding!
