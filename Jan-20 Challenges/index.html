Write a function that will
return the count of distinct
case-insensitive alphabetic
characters and numeric digits
that occur more than once in
the input string. The input
string can be assumed to contain
only alphabets (both uppercase and lowercase)
and numeric digits.




You are going to be given a word.
Your job is to return the middle
character of the word. If the word's
length is odd, return the middle
character. If the word's length is even,
return the middle 2 characters.
Note: You will only be running single
words through this function, not multi-word
strings.
