Challenge 1
Some people have been killed!
You have managed to narrow
the suspects down to just a
few. Luckily, you know every
person who those suspects have
seen on the day of the murders.


Given a dictionary with all the names of
the suspects and everyone that they have
seen on that day which may look like this:
{
'James': ['Jacob', 'Bill', 'Lucas'],
'Johnny': ['David', 'Kyle', 'Lucas'],
'Peter': ['Lucy', 'Kyle']
}
and also a list of the names of the dead people:
['Lucas', 'Bill']

return the name of the one killer.



Challenge 2
You are tasked with creating a function
that takes a variable 'language'(a string)
as a parameter and outputs a 'Welcome'
greeting in that language. It should
default to English if the language is not
in the database, or in the event of an
invalid input. Think of a way to store
the languages as a database (eg an object).
