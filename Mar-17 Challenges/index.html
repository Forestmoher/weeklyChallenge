Challenge 1
Write a function that takes in a string,
and returns a string incremented by 1.

If the string already ends with a number,
the number should be incremented by 1.
If the string does not end with a number,
the number 1 should be appended to the new string.

Examples:
Hello => Hello1
Hello1 => Hello2
Hello99 => Hello100

Challenge 2
Write a function that takes in an RGB decimal value,
and returns the hexadecimal representation of that color.
Valid decimal values for RGB are 0 - 255.

Examples:
rgbToHex(66,138,245) => #428af5
rgbToHex(255,255,255) => #ffffff
