Challenge 1
Write a function, which takes a non-negative
integer (seconds) as input and returns the time in a human-readable
format (HH:MM:SS)

HH = hours, padded to 2 digits, range: 00 - 99
MM = minutes, padded to 2 digits, range: 00 - 59
SS = seconds, padded to 2 digits, range: 00 - 59
The maximum time never exceeds 359999 seconds (99:59:59)

Example:
myFunction(86399) => 23:59:59
myFunction(30) => 00:00:30
Challenge 2 (aka the fizzbuzz test)
Write a program that prints the numbers from 1 to 100.
But for multiples of three print “Fizz” instead of the
number and for the multiples of five print “Buzz”.
For numbers that are multiples of both three and five print
“FizzBuzz”.
